Наберите в редакторе код из задания символ в символ и нажмите «Запустить».

```go
// Определение пакета
package main

// Импорт пакета
import "fmt"

// Определение функции main
func main() {
	// Вызов функции Print из пакета fmt
	// Отступ 1 таб
	fmt.Print("Hello, World!") // В конце не нужна точка с запятой
}
```
